# v95 - SCCB 회의록 Page 생성

## 추가 기능
- `회의록 Page 생성`은 이제 `이번주 회의록 URL`에 입력한 SCCB 회의록을 원본 Storage 형식으로 복제합니다.
- 회의록 제목의 주차/회의일을 다음 주로 변경합니다.
  - 예: `25W (06/22일) - 5건(A0, B3, C2)` → `26W (06/29일) - 5건(A0, B3, C2)`
- `1. SCCB 정보`의 `날짜/시간` 표에 들어 있는 Handy Date/표시 날짜를 다음 회의일로 변경합니다.
- `3. SCCB 리스트` 아래 Jira 매크로의 `SCCB 완료일 >=` / `SCCB 완료일 <=` 날짜를 7일 뒤로 이동합니다.
  - JQL 본문과 `RAW` 매크로 파라미터를 함께 변경합니다.
- 연도 전환 시 `YY년 회의록 (SCCB)` 상위 페이지를 같은 레벨에 새로 만들거나 기존 페이지를 재사용한 뒤, 제목을 `1W`부터 시작합니다.
- 동일 상위 페이지 아래 동일 제목이 이미 있으면 중복 생성하지 않습니다.

## UI 변경
- SCCB 주간 페이지 URL과 회의록 URL을 분리했습니다.
  - `SCCB Page 생성`: 이번주 SCCB URL 사용
  - `회의록 Page 생성`: 이번주 회의록 URL 사용
