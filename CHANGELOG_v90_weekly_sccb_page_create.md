# v90: SCCB Page 생성 — 매주 주간 Confluence 페이지 자동 생성

## 목적
Confluence 좌측 트리에 보이는 주간 SCCB 페이지를, 현재 주 페이지를 기준으로 다음 주 페이지로 자동 생성합니다.

## SCCB Page 생성 버튼 동작
1. `이번주 SCCB URL`에 현재 주간 SCCB 페이지 URL을 입력합니다.
2. `SCCB Page 생성`을 누릅니다.
3. 프로그램이 현재 페이지를 읽어 다음 주 페이지를 **원본과 같은 Space / 같은 상위 페이지 아래(같은 좌측 트리 레벨)**에 생성합니다.

## 자동 처리 항목
- 제목 주차/기간 변경
  - 예: `26W (06/22~06/29)` → `27W (06/29~07/06)`
- 본문 복사
  - 표, 매크로, 본문 내용을 storage 형식으로 복사
- 본문 내 주간 머리말 날짜 범위 변경
  - `2026. 6. 22. ~ 2026. 6. 29.`
  - `2026-06-22 ~ 2026-06-29`
  - `06/22 ~ 06/29`
  - 위와 같이 원본 주간 범위와 정확히 일치하는 표기만 다음 주 범위로 변경
- 중복 생성 방지
  - 같은 상위 페이지 아래에 이미 다음 주 제목의 페이지가 있으면 새로 만들지 않고 기존 페이지를 엽니다.
- SCCB Page 생성 성공 또는 기존 페이지 확인 후, 입력 URL을 새 주 페이지 URL로 자동 변경합니다.
  - 다음 주에도 같은 버튼을 바로 이어서 사용할 수 있습니다.
- 연말/연초 처리
  - 다음 시작일의 ISO 주차를 사용하므로 `52W/53W → 1W` 전환을 처리합니다.

## 유지 사항
- 첨부파일은 복사하지 않습니다.
- 이슈 행/평가 요청 데이터 등 본문 표의 상세 내용은 원본 페이지와 동일하게 복사됩니다.
- `회의록 Page 생성`은 기존 동작을 유지하며, SCCB URL 입력값을 자동 변경하지 않습니다.
