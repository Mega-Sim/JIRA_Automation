# v91: SCCB Page 생성 — 월~일 기간 및 연도별 상위 페이지 자동 분리

## 변경 사항

### 1. 주간 날짜 범위를 월요일~일요일로 고정
- 다음 주 페이지의 기간은 항상 **월요일부터 일요일까지 7일**로 생성됩니다.
- 예: 기존 `26W (06/22~06/29)`에서 생성 시
  - 변경 전: `27W (06/29~07/06)`
  - 변경 후: `27W (06/29~07/05)`
- 본문 안의 동일한 주간 날짜 범위도 같은 기준으로 바뀝니다.

### 2. 연말/연초 자동 분리
- 다음 주 시작일의 연도가 바뀌면, 원래 연도별 기록 페이지와 같은 레벨에 새 상위 페이지를 만듭니다.
- 예: 2026년 마지막 주 페이지에서 다음 주를 생성하는 경우
  1. `27년 사전 SCCB 기록` 페이지가 있는지 확인합니다.
  2. 없으면 새로 생성합니다.
  3. `1W (01/04~01/10)` 페이지를 그 하위에 생성합니다.
- 이미 `27년 사전 SCCB 기록`이 있으면 재사용하며 중복 생성하지 않습니다.
