# v96 — 회의록 URL 입력 팝업

## 변경 사항

- 메인 화면에서 `이번주 회의록 URL` 입력란을 제거했습니다.
- 메인 화면에는 `이번주 SCCB URL` 입력란만 유지합니다.
- `회의록 Page 생성` 버튼을 누르면 `회의록 원본 URL 입력` 팝업이 열립니다.
- 팝업에서 URL을 입력한 뒤 **확인** 버튼 또는 **Enter**를 누르면 다음 주 회의록 생성이 시작됩니다.
- 빈 URL은 경고 후 팝업을 유지하며, Esc/취소/창 닫기로 생성 작업을 취소할 수 있습니다.

## 유지 사항

- 회의록 Storage 포맷 복제, 제목/날짜/시간 표 날짜 이동, SCCB 완료일 JQL 범위 이동, 연도별 상위 페이지 처리, 중복 생성 방지 로직은 v95와 동일합니다.
